# Transpiler

Implementação de um compilador com as seguintes funcionalidades: 
*Análise Léxica; 
*Análise Sintática; 
*Análise Semântica;
*Gerador de código; 

O compilador deve ser capaz de trabalhar com códigos que contenham: 
*Declaração e atribuição de variáveis; 
*Entrada e Saída de Dados; 
*Repetição; 
*Condicional;
